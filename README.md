🚀 Quick Start

Get your first digital agent up and running in minutes.

1.  Installation
    # Install via pip
    pip install proclaw

    # Or clone and install from source
    git clone https://github.com/your-org/proclaw.git
    cd proclaw
    pip install -e .
    

2.  Setup & Authentication
    # Run the setup wizard
    proclaw init

    # Follow the prompts to configure your environment and API keys.
    

3.  Run Your First Agent
    Create a simple my_first_agent.py:
    from proclaw import Agent, Skill

    # Load a research skill from the HUB
    market_research = Skill.from_hub("advanced-market-analysis")

    # Define your agent's mission
    @agent(mission="Research the renewable energy tech landscape for Q2 2026")
    def research_agent():
        data = market_research.execute(topic="solid-state batteries")
        return data.generate_report(format="md")

    # Deploy it
    if __name__ == "__main__":
        result = research_agent.run()
        print(result)
    
    Run it: python my_first_agent.py
